# 摇杆参数配置

（摘自教程置顶评论，建议看过 [教程](https://www.bilibili.com/video/BV1FksrzAEo7) 后完全理解后查阅使用）

补充一下摆放细节，在碰撞盒和碰撞触发器全部使用默认正方体（长宽高均为1）的情况下：

1. 额外碰撞盒，前后两个的位置在±0.6，左右两个位置在±0.7，这样角色所在的空间尺寸为0.2×0.4，刚好能够左右微移
2. 碰撞触发器，左右位置分别为±1.19，刚好能蹭到角色活动空间的边缘
3. 力场器，内径调成0.25，外径调成0.35，方向朝内，速度5
