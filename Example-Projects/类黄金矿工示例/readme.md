# 类黄金矿工玩法示例地图

> 基于高级功能模块库实现的黄金矿工风格玩法，展示负载感知和多态设计的实际应用

## 🎯 玩法介绍

### 核心机制

- **钩爪发射**：瞄准并发射钩爪捕获矿物
- **资源收集**：收集不同价值的矿物获得分数
- **时间挑战**：在限定时间内获得最高分数
- **障碍规避**：避开岩石等障碍物

### 特色功能

- **智能负载管理**：流畅的物理计算和渲染
- **多态矿物系统**：不同矿物具有独特行为和价值
- **简洁控制接口**：易于理解和扩展的输入系统
