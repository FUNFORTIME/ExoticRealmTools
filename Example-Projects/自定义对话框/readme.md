

# 对话框模板地图

> 一个为《原神》「千星奇域」玩法设计的自定义对话框模板，助你轻松为你的关卡创建丰富的剧情和对话。

## 🗺️ 模板简介

本模板地图预先配置了一套完整的对话框系统，包含了多种对话场景的示例。你可以直接使用或基于此模板进行修改，快速为你创建的关卡添加剧情文本、任务指引和角色互动。

## ⚠️ 重要前提与使用说明

### **奇匠等级要求**
- **确认你的奇匠等级至少达到 2 级**，否则将无法正常显示自定义文本内容。
- 建议在下载和使用前，先在游戏内提升你的奇匠等级。

### **模板内容**
- 地图中已内置 **示例节点图**，清晰展示了对话框的完整逻辑流程。
- 你可以通过参考和修改这些示例节点，来理解并创建属于你自己的对话。

## 🚀 如何使用

1.  **下载存档**：从上方链接下载存档文件（并解压）。
2.  **导入游戏**：在「千星沙箱」编辑器中，找到并导入此存档文件。
3.  **参考学习**：在编辑器中打开，仔细研究“示例节点图”的连接与配置方式。
4.  **自定义修改**：根据你的关卡需求，修改对话文本、触发条件和角色名称等。
5.  **测试与发布**：在游戏内测试对话触发是否正常，确认无误后即可发布你的地图。
