# 原神千星奇域 - 创作者工具与资源库

> 本仓库是为《原神》「千星奇域」UGC 玩法建立的资源集合，旨在帮助创作者更高效地利用官方编辑器「千星沙箱」进行关卡创作与玩法设计。

## ✨ 项目简介

「千星奇域」是《原神》在「月之二」版本中推出的UGC玩法，它提供了一个名为**千星沙箱**的图形化编辑器，允许创作者调用《原神》中几乎全量的玩法功能与资产，自由构建从大富翁、Roguelike到密室逃脱、塔防等多样化的玩法地图。

本仓库致力于收集和整理与「千星奇域」开发相关的工具、脚本、组件与数据，助力创作者降低开发门槛，释放无限创意。

源项目位于Github，Gitee的项目从其拉取。

## 🗂️ 资源目录


- **`Online/`**：存放在线工具链接（`.md`文件）。[于此跳转](./Online)

- **`Documentation/`**：存放创作者经验文档（`.md`文件），[于此跳转](./Documentation)，如新手指南、高级技巧、问题排查等。

- **`Example-Projects/`**：存放完整的示例项目。[于此跳转](./Example-Projects/%E7%A4%BA%E4%BE%8B%E5%AD%98%E6%A1%A3.md)，每个项目是一个文件夹，内含项目`存档文件`和说明文档`README.md`。

- **`Scripts/`**：存放脚本和节点图。[于此跳转](./Scripts)，其下按功能分设子文件夹（如`game-mechanics/`），每个子文件夹内含`节点图文件`和说明文档`README.md`。

- **`Prefab/`**：存放预制元件。[于此跳转](./Prefab/%E6%A8%A1%E6%9D%BF%E5%85%83%E4%BB%B6.md)，其下按类型分设子文件夹（如`interactive-elements/`），每个子文件夹内含`元件资源`和说明文档`README.md`。


## 🤝 为社区贡献力量

我们欢迎所有创作者分享你的资源、教程和心得！你可以通过以下方式参与：

### 📝 分享创作经验
将你的经验总结成文档，提交到 `Documentation/` 目录，帮助其他创作者少走弯路。
- 最佳实践指南
- 疑难问题解决方案
- 创作技巧与心得
- 直接提交markdown文档即可

### 📜 贡献脚本组件
将你封装好的通用逻辑脚本提交到 `Scripts/` 目录。
- 可复用的节点图文件
- 通用功能逻辑脚本
- 效率工具脚本

### 🧩 分享预制元件
将你制作的实用元件提交到 `Prefab/` 目录。
- 常用机关元件
- 场景装饰组件
- 交互功能模块
- 「月之三」版本预计会更新元件库，此后会收录一些较为泛用的元件GUID供查找。

### 🎮 提交示例项目
在 `Example-Projects/` 中分享你的关卡工程文件，作为学习范例。
- 完整关卡存档
- 教学演示项目
- 创意玩法原型

**贡献指南：**
1. 请在对应目录下创建清晰的文件夹命名
2. 每个提交请包含 `README.md` 说明文件
3. 确保内容原创或已获得授权
4. 遵循清晰的目录结构规范

（*注：请确保所分享的内容不侵犯米哈游的著作权，并遵循《原神》用户协议。*）
