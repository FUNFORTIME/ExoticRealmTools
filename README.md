# 原神千星奇域 - 创作者工具与资源库

> 本仓库是为《原神》「千星奇域」UGC 玩法建立的资源集合，旨在帮助创作者更高效地利用官方编辑器「千星沙箱」进行关卡创作与玩法设计。

## ✨ 项目简介

「千星奇域」是《原神》在「月之二」版本中推出的UGC玩法，它提供了一个名为**千星沙箱**的图形化编辑器，允许创作者调用《原神》中几乎全量的玩法功能与资产，自由构建从大富翁、Roguelike到密室逃脱、塔防等多样化的玩法地图。

本仓库致力于收集和整理与「千星奇域」开发相关的工具、脚本、组件与数据，助力创作者降低开发门槛，释放无限创意。

## 🤝 为社区贡献力量

我们欢迎所有创作者分享你的资源、教程和心得！你可以通过以下方式参与：
1.  **分享你的组件**：将你封装好的通用逻辑脚本或组件提交到 `Scripts-And-Components`。
2.  **贡献教程**：将你的经验总结成文档，帮助其他创作者少走弯路。
3.  **提交案例**：在 `Example-Projects` 中分享你的关卡工程文件（如适用），作为学习范例。

（*注：请确保所分享的内容不侵犯米哈游的著作权，并遵循《原神》用户协议。*）
