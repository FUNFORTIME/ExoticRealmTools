# 原神千星奇域 - 创作者工具与资源库

> 本仓库是为《原神》「千星奇域」UGC 玩法建立的资源集合，旨在帮助创作者更高效地利用官方编辑器「千星沙箱」进行关卡创作与玩法设计。

## ✨ 项目简介

本仓库致力于收集和整理与「千星奇域」开发相关的工具、脚本、组件与数据，助力创作者降低开发门槛，释放无限创意。

源项目位于Github，Gitee的项目从其拉取。

现在仓库正在执行结构更新，首页说明内容正在完善。

## 🗂️ 资源目录


- **`基础功能/`**：实现基础的节点图功能，通常不构成完整模块。[于此跳转](./%E5%9F%BA%E7%A1%80%E5%8A%9F%E8%83%BD)

- **`在线工具/`**：存放在线工具链接。[于此跳转](./%E5%9C%A8%E7%BA%BF%E5%B7%A5%E5%85%B7)

- **`示例项目/`**：存放完整的示例项目。[于此跳转](./Example-Projects/%E7%A4%BA%E4%BE%8B%E5%AD%98%E6%A1%A3%E7%B4%A2%E5%BC%95.md)
